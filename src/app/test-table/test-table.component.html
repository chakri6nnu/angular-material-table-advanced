Data Table Library Test Component

This component demonstrates the data-table-lib with comprehensive configuration options.

Configuration Features:

  • ✅ Column widths (ID: 80px, VIN: 200px, Brand: 150px, Year: 100px, Color: 120px)
  • ✅ Sorting enabled with default sort on ID column
  • ✅ Filtering enabled (global search + column filters)
  • ✅ Pagination enabled (default: 10 items per page, options: 5, 10, 25, 50, 100)
  • ✅ Grouping enabled with default group by Brand
  • ✅ Selection enabled (multi-select)
  • ✅ Export to CSV enabled
  • ✅ Column visibility toggle enabled
